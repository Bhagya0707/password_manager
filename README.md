# Password Manager
**Tech Stack:** Python, Tkinter
**Description:** A simple Python GUI application to generate and save strong passwords for websites.
**Features:** Generate strong passwords, Save passwords in a text file, View saved passwords in console
**How to Run:** 
1. Open Command Prompt 
2. Navigate to the frontend folder: cd "C:\Users\ADMIN\OneDrive\Desktop\password_manager\frontend" 
3. Run the Python GUI: python password_gui.py 
4. Enter a website name, click Generate Password, then click Save Password 
5. Open passwords.txt to see the saved passwords
**Project Files:** frontend/password_gui.py → Main Python GUI code, passwords.txt → Stores saved passwords, README.md → This file
